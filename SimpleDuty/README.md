# SimpleDuty

SimpleDuty是一个简单的ClassIsland插件，用于显示每日轮换的值日生信息。

## 功能特性

- 自动根据设置的参与轮换天数切换值日生
- 支持自定义显示格式（{n}表示编号，{s}表示姓名）
- 可配置参与轮换的星期几、基准日期和偏移量
- 支持导入和编辑值日生名单


## 使用说明

1. 在ClassIsland中添加值日生组件到主界面
2. 配置参与轮换的星期几（周一到周六/周一到周五/每天）
3. 设置自定义显示格式，例如：`值日：{n}{s}`
4. 导入或编辑值日生名单（每行一个，格式：编号 姓名）
5. 配置基准日期和偏移量调整当前值日生
